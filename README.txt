The scripts and data in this repository correspond to the following publication:

BR Jack, DR Boutz, ML Paff, BL Smith, JJ Bull, CO Wilke (2017). Reduced protein expression in a virus attenuated by codon deoptimization. G3 (Bethesda). DOI: 10.1534/g3.117.041020 

Raw data is not available in this repository. Please see the GEO database (ID: GSE96573) and ProteomeXchange Consortium via the PRIDE partner repository (ID: PXD006502) to access the raw data.

data/ - all post-processed data
figures/ - figures that appear in publication in PDF format
rmarkdown/ - proteomics and rna-seq analyses and plots
scripts/ - R and python scripts for data processing and some plots
